Compose Css3 Classes and Animations

Cautions:p>

  • The SQL database created by Visual Studio 2022 during installation is used.
    *Other SQL databases may be used instead of localdb by setting the value of the registry key 'HKEY_CURRENT_USER\SOFTWARE\_ZEN_SOFT\CssScriptClassesExt' to a connection string for another database where permission exists. **I have only tested with other databases on my local machine. (The connection string I used is shown in the image at the bottom of this page )
  • The first time the extension starts it creates a data base for Css3 components in classes, please wait until it is done before using it. When the settings box appears it is done.
  • Sometimes the script displayed does not update, press the 'Build Style' button to update the window.
  • If all of the extension tools do not show in the window when the first time it is run, close the extension window and restart using the Tool menu entry for Css_Classes
  • Please let me know of bugs.

Recent changes

  • This is the version for Visual studio 2022

The Css Script Classes Extension for Visual Studio enables composing Css Classes and animation key frames by selecting their components and it saves the components in the (localdb) database created when visual studio 2015 is installed so that users have copies to be used or re-edited as needed.

The tool enables selection of components in Css3 classes and Animation Key Frames, It allows selection of colors in gradients

Selection is done from expanding controls in the settings box

Css3 style classes may be renamed in the tool

Pressing the 'copy' button creates a copy

When you have a lot of Css3 classes, the 'Choose Style button is useful.

Pressing 'Compose/View Animation brings up the page where animation key frames are edited

Animation may be mixed into styles by selecting a set of key frames by name

Setting a registry value allows using a different SQL database

Thank you for using this extension.
